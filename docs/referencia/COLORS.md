# Paleta de Colores - Panorama Ingeniería

## Colores Principales

### Azul Navy (Principal)
- **Hex**: `#023047`
- **RGB**: `rgb(2, 48, 71)`
- **Uso**: Fondos principales, textos importantes, elementos de navegación
- **Clase Tailwind**: `bg-panorama-navy`, `text-panorama-navy`

### Azul Cielo (Sky)
- **Hex**: `#8ECAE6`
- **RGB**: `rgb(142, 202, 230)`
- **Uso**: Acentos, highlights, elementos secundarios
- **Clase Tailwind**: `bg-panorama-sky`, `text-panorama-sky`

### Naranja (Acción)
- **Hex**: `#FF5E13`
- **RGB**: `rgb(255, 94, 19)`
- **Uso**: Botones de acción, CTAs, elementos destacados
- **Clase Tailwind**: `bg-panorama-orange`, `text-panorama-orange`

### Rosa/Magenta (Acento)
- **Hex**: `#FE0467`
- **RGB**: `rgb(254, 4, 103)`
- **Uso**: Acentos especiales, elementos de énfasis
- **Clase Tailwind**: `bg-panorama-pink`, `text-panorama-pink`

### Texto Oscuro
- **Hex**: `#0B1320`
- **RGB**: `rgb(11, 19, 32)`
- **Uso**: Textos principales, títulos
- **Clase Tailwind**: `text-panorama-dark`

## Escalas de Color

Cada color tiene una escala del 50 al 900 para diferentes niveles de intensidad:
- `50`: Más claro
- `500`: Color principal (DEFAULT)
- `900`: Más oscuro

## Ejemplos de Uso

```jsx
// Fondo azul navy
<div className="bg-panorama-navy text-white">

// Botón naranja
<button className="bg-panorama-orange hover:bg-panorama-orange-600">

// Texto azul cielo
<p className="text-panorama-sky">

// Borde rosa
<div className="border-2 border-panorama-pink">
```
